# India COVID-19 Dashboard Analysis

An interactive Tableau Public dashboard analyzing COVID-19 trends across India using real-world datasets from Kaggle.

## 🔗 Live Dashboard
[Click here to view the dashboard](https://public.tableau.com/app/profile/vidapanakal.swetha/viz/IndiaCOVID-19Dashboard_17821347543960/Dashboard2)

## 📊 Charts Included
- Total Deaths by State (India Map)
- Age Group Distribution (Pie Chart)
- Gender Distribution (Pie Chart)
- Statewise Confirmed Cases (Bar Chart)
- ICMR Testing Labs by State (Bar Chart)
- Statewise Testing Details (Bar Chart)
- First vs Second Dose Administered (Bar Chart)
- Vaccination Progress by State (Bar Chart)

## 🛠️ Tools Used
- Tableau Public 2026
- Microsoft Excel (data cleaning)

## 📁 Dataset Source
- [Kaggle — COVID-19 India Dataset](https://www.kaggle.com/)

## 📌 Key Insights
- Maharashtra recorded the highest COVID-19 deaths among all states
- Covishield was the most administered vaccine across India
- Male cases were significantly higher than female cases
- Uttar Pradesh had the highest number of total tests conducted
